available array [m]
  - m = number of types of resources
  - indicates number of available resources of each type

max demand [n][m]
  - n = number of processes
  - defines maximum demand of each resource by each processes

allocation [n][m]
  - defines number of each resource allocated to each process

need [n][m]
  - defines number of each resource still needed by each process

Baneker's Algorithm runs every time a process puts forth a request fora resource.
